# Publishing Steps

Repository:

https://github.com/mikeyloops-glitch/pressureflip-solutions

Live site once published:

https://mikeyloops-glitch.github.io/pressureflip-solutions/

## Browser upload method

1. Open the repository on GitHub.
2. Click `Add file`.
3. Click `Upload files`.
4. Drag these items from this folder into GitHub:
   - `index.html`
   - `styles.css`
   - `README.md`
   - `research-notes.md`
   - `.nojekyll`
   - the `assets` folder
5. Commit directly to `main`.
6. Open repository `Settings`.
7. Open `Pages`.
8. Choose `Deploy from a branch`.
9. Select branch `main` and folder `/root`.
10. Click `Save`.

GitHub normally publishes the site within a few minutes.
