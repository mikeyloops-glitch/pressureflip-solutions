# PressureFlip Solutions

Industrial website draft for PressureFlip Solutions, LEL instrumentation specialists and agents for NIRA Instruments in South Africa.

## Files

- `index.html` - the website page.
- `styles.css` - visual design and responsive layout.
- `assets/pressureflip-logo.svg` - recreated logo from the uploaded reference.
- `assets/pressureflip-systems-visual.svg` - technical hero visual.
- `research-notes.md` - source notes used for positioning and copy.
